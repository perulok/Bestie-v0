Bestie-chat-v0 (static landing)
--------------------------------

How to run (no backend needed):
1) Open 'index.html' in your browser (double click).
   - or start a local server: `python3 -m http.server 3000` and go to http://localhost:3000

What you get:
- Minimal landing like OpenAI/Anthropic/Perplexity.
- Logo + tagline + single prompt input + suggestions.
- A placeholder card echoes your prompt (replace with your API later).

Where to connect your backend:
- In index.html, search for 'TODO: replace with your backend call'.
